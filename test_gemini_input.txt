የተለያዩ የቅድመ ጥገና ስራ ለማከናወን ሲባል በዕቅድ የሚቋረጥ የኃይል አቅርቦት
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp; &nbsp; &nbsp; &nbsp; የተለያዩ የቅድመ ጥገና የመልሶ ግንባታ ስራ ለማከናወን ሲባል በዕቅድ የሚቋረጥ የኃይል አቅርቦት&nbsp; ተቁ ኃይልየሚቋረጥበት ሪጅን እና ከተማ የሚቋረጥበት ቀን የሚቋረጥበት ምክንያት አገልግሎቱ የሚቋረጥባቸው አካባቢዎች የሚቋረጥበት የቆይታ ጊዜ &nbsp; &nbsp; &nbsp; &nbsp; 1 &nbsp; &nbsp; &nbsp; &nbsp; አዲስ አበባ &nbsp; &nbsp; &nbsp; መጋቢት 8 ቀን 2017 ዓ.ም &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; መጋቢት 9 ቀን 2017 ዓ.ም &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; የመልሶ ግንባታ ስራ ለማከናወን ጉርድ ሾላ ቴሌ፣ ትምህርት መሳሪያዎች ማምረቻ ድርጅት፣ ግብርና ምርምር ፣ ጃክሮስ አደባባይ በስተጀርባ፣አየር መንገድ ዴዲኬትድትደድ፣ጀሞነ&nbsp; -2 ኮንዶሚኒየም ፣ የስ ውሃ ፣ዓለምገና ማርስ ሕንጻ፣በኮተቤ፣ &nbsp; &nbsp; &nbsp; ከጠዋቱ 2፡30 እስከ ቀን 9፡30 ሰዓት &nbsp; 2 &nbsp; &nbsp; &nbsp; አዲስ አበባ&nbsp; መብራት ኃይል ጋራዥ ጀርባ፣ ት/ቤት፣ሳሊተ ምሕረት አካባቢ፣ ጆርጅ ዘይት ቤት ፣ ሜታ ቢራ አካባቢ፣ &nbsp; &nbsp; ከጠዋቱ 2፡30 እስከ ቀን 9፡00 ሰዓት &nbsp; 3 &nbsp; &nbsp; አዲስ አበባ &nbsp; መጋቢት 10 ቀን 2017 ዓ.ም &nbsp; &nbsp; የመልሶ ግንባታ ስራ ለማከናወን ከኮተቤ ብረታ ብረት፣ ኮተቤ ኪዳነምህረት ቤ/ክ አካባቢ፣ አንቆርጫ ሰፈር &nbsp; &nbsp; ከጠዋቱ 2፡00 እስከ ቀን 10፡00 ሰዓት &nbsp; 4 &nbsp; ደብረማርቆስ&nbsp; &nbsp; መጋቢት 8 ቀን 2017 ዓ.ም &nbsp; የመልሶ ግንባታ ስራ ለማከናወን ጎፍጭማ፣ዋበር፣አንጂም፣ዪኒቨርስቲ፣ኢንዱስትሪ፣ደ/ማርቆስ ውሃ፣ሉማሜ እና አማኑኤል &nbsp;ባታ፣ኢንዱስትሪ በከፊልሏ፣ ስላሴ፣አባይኩኑ ኮንዶሚኒየም፣አብማት/ቤት፣አብማቤ/ክ፣አዲሱ ፍ/ቤት ፣ አባ ብዙነህ ጉርድ፣ ገጠር መንገድ ኮንዶሚኒየም፣የተባበሩት፣አሮጌውሆስፒታልኮንዶሚኒየም፣ቤተ መንግስት፣ድብዛ ት/ቤት ፣5 ኪሎ፣ቦሌኮንዶሚኒየም፣እነራታ፣የጸድ፣ሮብገበያ፣ሊባኖስ፣ወንቃ፣ቸርተከል፣ፈንድቃ፣የውላ፣ደኤሊያስ፣ግራቅዳምን፣ደጋሰኝን፣አማኑኤል፣ደንባ፣ጭምት፣ጓይ፣ጎፍጭማ፣ኳሽባ ገዳም፣የጁቤ፣አምበር፣ሉማሜ &nbsp;
